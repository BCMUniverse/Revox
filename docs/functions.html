Funções incluídas no Revox

htmlText.h

  • CreateTag: Cria os elementos
  • SubString: copia substrings de uma string
  • SubString2: copia substring e remove-as de uma string
  • InitHTMLText: Analisa o Documento ou arquivo recebido e converte o conteúdo HTML

http.h

  • recvtimeout: recebe pacotes vindo da web
  • InitHTTP: recebe e acessa endereço recebido

revox.h

  • InitGumbo: analisa documento html
  • InitText: inicia o analisador html e acessa endereço recebido
  • InitFileText: analisa um arquivo html

socket.h

  • InitSocket: recebe sockets a partir de um endereço

Implementando as Funções

htmlText.h

  • CreateTag: char *CreateTag(char *inicio, char *tag, char *fim)
  • SubString: char *SubString(char *str, int inicio, int fim)
  • SubString2: char *SubString2(char *str, char *inicio, char *fim)
  • InitHTMLText: char *InitHTMLText(char *content)

http.h

  • recvtimeout: int recvtimeout(int s, int timeout)
  • InitHTTP: char *InitHTTP(char *address, int port, char *caminho, char *cookie)

revox.h

  • InitGumbo: void InitGumbo(char *host)
  • InitText: void InitText(char *host)
  • InitFileText: void InitFileText(char *path)

socket.h

  • InitSocket: void InitSocket(char *hostname)

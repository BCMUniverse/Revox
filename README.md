BCM Revox Engine <http://revox.sf.net/>

##O que é Revox?

Revox é um renderizador web, escrito do zero por Cleber Matheus, que ler arquivos HTML e converte-os em páginas web.

O Revox acessa os arquivos HTML no disco local ou através do protocolo HTTP e outros protocolos da internet, lê os arquivos recebidos e interpreta-os segundo o padrão definido pela Engine(futuramente deverá interpretar conforme os padrões do HTML5.1).
